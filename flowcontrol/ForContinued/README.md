### For continued
```go
sum := 1
for ; sum < 1000; {
  sum += sum
}
```
`초기화`와 `증감문`은 생략해도 된다.<br>

### For is Go's "while"
```go
sum := 1
for sum < 1000 {
  sum += sum
}
```
`;`을 생략해도 된다.<br>
