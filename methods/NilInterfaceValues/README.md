### Nil interface values
`nil interface`는 값과 `concrete type` 모두 가지고 있지 않다.<br>
`nil interface`에서 `method`를 부르는 것은 `run-time error`다.<br>
왜냐하면, `interface tuple`에서 호출할 `concrete method`가 없기 때문이다.<br>
