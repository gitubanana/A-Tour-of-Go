### Interface values
내부적으론, `interface`는 값과 `concrete type`의 튜플로 생각할 수 있다.<br>
```go
(value, type)
```

`interface`는 `concrete type`의 값을 가지고 있다.<br>
`interface`에서 `method`를 부르는 것은 `concrete type`의 `method`를 부르는 것과 같다.<br>
