### Variables with initializers
`var`는 각 변수마다 초기화를 할 수 있다.<br>
초기화를 한다면, 타입은 생략 가능하다. (변수는 초깃값의 타입으로 설정된다.)<br>
```go
var c, python, java = true, false, "no!"
```
