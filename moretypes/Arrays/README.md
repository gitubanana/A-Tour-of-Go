### Arrays
```go
var a [10]int // 10 크기의 int 배열 a
```
`[n]T`는 `n` 크기의 원소 타입이 `T`인 배열이다.<br>
> [!NOTE]
> 배열의 크기도 타입의 일부분이기 때문에, 배열의 크기는 변동될 수 없다.<br>
> 하지만, `Go`에는 배열을 다루는 편리한 방법이 있어서 걱정하지 말래요~<br>
