### Slice literals
`slice literal`은 크기가 없는 `array literal` 같다.<br>
- **array literal**<br>
  ```go
  [3]bool{true, true, false}
  ```
- **slice literal**<br>
  ```go
  []bool{true, true, false}
  ```
  `array literal`처럼 배열을 만들고, 그 배열을 참조하는 `slice`를 만든다.<br>
